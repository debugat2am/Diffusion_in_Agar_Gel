#  Diffusion in Agar Gel using Fick's Law

##  Project Overview

This project investigates the diffusion of dye through agar gel under different concentrations and temperature conditions. The experiment is based on **Fick's Law of Diffusion**, and MATLAB was used to analyze the experimental data, calculate diffusion coefficients, and visualize diffusion trends.

---

##  Objectives

- Study the diffusion behavior of dye in agar gel.
- Compare diffusion at different agar concentrations and temperatures.
- Calculate diffusion coefficients using Fick's Law.
- Analyze experimental data using MATLAB.
- Visualize diffusion trends through graphs.

---

##  Tools & Technologies

- MATLAB R2022b
- Microsoft Excel
- Fick's Law of Diffusion
- Chemical Engineering Laboratory
